An número de cor is um ColorRef. 
\[Exemplo: $AaBbGgRr onde An é o Alfa, B é o Azul, G é o verde e R o vermelho]
\[basicamente é o mesmo que um código hexadecimal HTML, mas ao contrário]
\Por exemplo:
\ A cor vermelha em código hexadecimal HTML é #ff0000 
\ A cor vermelha em ColorRef é $000000FF
\ A cor vermelha verde claro em código hexadecimal HTML é #00ff00
\ A cor verde claro em ColorRef É $0000FF00
\ A cor vermelha verde escuro em código hexadecimal HTML é #008000
\ A cor verde escuro em ColorRef É $00008000
\ ColorRef Azul = $00FF0000
\ ColorRef Preto = $00000000
\ ColorRef Branco = $00FFFFFF

O número da cor de arco-íris é um número.

\\A saturação é um número. [que vai de 0 até 1000]
