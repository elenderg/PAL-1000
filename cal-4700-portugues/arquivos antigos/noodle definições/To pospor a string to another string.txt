To pospor a string to another string:
If the string is blank, exit.
Colocar the string's comprimento into a comprimento combinado.
Colocar the outro string's comprimento into a comprimento salvo.
Adicionar the comprimento salvo to the comprimento combinado.
Reatribuir the outro string's first given the comprimento combinado.
Colocar the outro string's first plus the comprimento salvo into a pointer.
Copiar bytes from the string's first to the pointer for the string's comprimento.
Colocar the outro string's first plus the comprimento combinado minus 1 into the other string's last.
