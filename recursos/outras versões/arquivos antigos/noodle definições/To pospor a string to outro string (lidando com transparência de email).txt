To pospor a string to outro string (lidando com transparência de email):
If the string is blank, exit.
Mandar a rider on the string.
Percorrer.
Mover the rider (text file rules).
If the rider's token is blank, exit.
If the rider's token starts with ".", pospor "." to the outra string.
Pospor the rider's token to the other string.
Repetir.
