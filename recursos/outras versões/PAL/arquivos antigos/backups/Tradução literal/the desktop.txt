Se o dialog's caixa de texto não existir, deixe quieto.
Se o event não existir, deixe quieto.
Extend any selection in o dialog's caixa de texto usando o event's local.
Se o event's shift flag is not set, deselect o dialog's caixa de texto.
Clear uma selection.
Percorra.
If uma flag estiver verde, show o ambiente usando estado.
Se o mouse's left botão is up, deixe quieto.
Coloque o mouse's local dentro de um local.
Autoscroll o dialog's caixa de texto usando o local e o flag.
Extend any selection in o dialog's caixa de texto usando o local.
Se o dialog's caixa de texto's selection is o selection, repeat.
Mude para a cor verde flag.
Coloque o dialog's caixa de texto's selection no selection.
Repita.

Como track uma aba returning uma flag:
Mude para uma cor vermelha flag.
Se a aba não existir, deixe quieto.
Percorra.
Se o mouse's left botão is up, deixe quieto.
Coloque o mouse's local dentro de um local.
Encontre outra aba usando o local.
Se o outro aba não existir, repeat.
Se a aba is o outro aba, repeat.
Mude para a cor verde flag.
Remove a aba usando a abas.
Se o outro aba's left is less than a aba's left, insert a aba na abas before o outro aba.
Se o outro aba's left is greater than a aba's left, insert a aba na abas after o outro aba.
Adjust a abas.
Show o ambiente.
Repita.
